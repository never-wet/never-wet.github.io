# Project Memory

This folder stores compact assistant-facing project memory for future coding sessions.

It is intentionally separate from `src/memory/`:

- `src/memory/` is runtime source-of-truth for the app itself.
- `memory/` is handoff context for humans and coding assistants.

## Read Order

1. `project-context.md`
2. `design-direction.md`
3. `recent-changes.md`

## Goal

Keep the current product direction, important decisions, and recent implementation context in a few small files so future work does not require reading large parts of the codebase first.
