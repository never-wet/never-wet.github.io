# Model Attribution

This project reuses the repository's existing user-provided GLB vehicle assets.

- Porsche 911 GT3: `porsche_911_gt3.glb`. The original download dialog credits Outlaw Games and lists Creative Commons Attribution-NonCommercial.
- Ferrari SF90 Spider: `2021_ferrari_sf90_spider.glb`.
- Lotus Emira 2022: `lotus_emira_2022__www.vecarz.com.glb`.
- BMW M3 Competition G80: `2021_bmw_m3_competition_g80.glb`.

These files are mirrored in `public/models/` for local Next.js development and copied into `out/models/` by the static export.
