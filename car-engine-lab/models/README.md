# Models

This folder is reserved for future GLB or compressed mesh assets.

The current lab uses procedural Three.js/R3F engine assemblies so the static site loads quickly and still shows realistic moving internal structures:

- cylinder banks
- pistons
- connecting rods
- crankshafts
- valves
- rotary housing and rotor
- electric stator and rotor
- turbocharger and supercharger hardware
