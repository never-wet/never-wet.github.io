# Icon System

The three main chamber structures are now emblem sculptures tied to each project identity.

## Lantern Oath

- icon: lantern sigil
- cues: warm brass ring, smoked glass lantern body, ember core
- purpose: read immediately as mythic RPG / hearth / exploration

## Midnight Pawn

- icon: broker pawn
- cues: pawn silhouette, opened ledger base, merchant token ring
- purpose: read as a stylized broker / pawn / vault emblem instead of a random box

## Reels Pull

- icon: reel machine
- cues: slot-machine body, three reel discs, lever, orbit ring
- purpose: read instantly as a game of reels, pulls, and reward loops
