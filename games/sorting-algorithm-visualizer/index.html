Algorithm

Quick Sort

Partitions around a pivot so smaller values go left and larger values go right.

Best
O(n log n)
Average
O(n log n)
Worst
O(n^2)
Space
O(log n)
Stable
No
Use Case
Fast general-purpose sorting when stable order is not required.

Pseudocode

Quick Sort

  1. 01choose a pivot in the active range
  2. 02scan values before the pivot
  3. 03if a value belongs on the left side
  4. 04 swap it into the left partition
  5. 05place the pivot between both partitions
  6. 06sort the left and right ranges
