# Stitch Soccer Clicker Tycoon

This folder is a soccer clicker / tycoon concept bundle rather than one single finished game entry.

## What is here

- `the_pitch_soccer_clicker/`
  Main clicker / field concept screen.
- `upgrade_shop/`
  Upgrade shop concept screen.
- `global_leaderboard/`
  Leaderboard concept screen.
- `player_profile/`
  Player profile concept screen.
- `pitch_power/`
  Design-system and direction docs for the project.

## Purpose

This folder looks like a collection of prototype screens and design exploration for a larger soccer game experience. It is useful as:

- UI reference material
- layout experiments
- visual direction / design system work
- source material for folding pieces into a more complete future game

## Helpful files

- `pitch_power/DESIGN.md`
  Design language, palette, typography, and component ideas.
- `*/code.html`
  Standalone concept pages for specific parts of the product.

## Notes

- This folder does not currently present itself like one single root-playable game.
- If you want this project to behave like the other games under `/games`, the next good step would be adding a root `index.html` that links these prototype screens together.
