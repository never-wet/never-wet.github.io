# Soccer Idle

`Soccer Idle` is a browser-based idle / clicker game built around a soccer club theme.

## What it does

- Lets the player earn money by tapping and through passive income
- Includes multiple screens like field, upgrades, squad, leaderboard, stats, and more
- Uses a more styled app-like layout than the main arcade page
- Saves progress in the browser

## Main files

- `index.html`
  The main app shell and UI layout.
- `script.js`
  The game logic, screen switching, upgrades, economy, and save behavior.
- `styles.css`
  The main visual styling for the game.

## Run it

Open:

- `/games/soccer-idle/`

Or locally:

- `games/soccer-idle/index.html`

## Notes for editing

- This game is split more traditionally across HTML, CSS, and JS files.
- The in-browser save system is important to preserve when refactoring game logic.
